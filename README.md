# DSA Practice

Daily data structures and algorithms practice in C++.

## Overview

This repository collects practice implementations and LeetCode-style solutions for trees, linked lists, hashing, and bit manipulation. The goal is to keep each solution easy to read, compile, and revisit during interview preparation.

## Structure

```text
trees/                         Binary tree and BST practice
linked list implementation/    Singly and circular linked list examples
hashing/                       Hash map and string problems
LeetCode/Binary_trees/         Binary tree LeetCode solutions
LeetCode/single_linked_list/   Linked list LeetCode solutions
LeetCode/daily_ task/          Daily challenge solutions
```

## Topics Covered

| Topic | Examples |
| --- | --- |
| Trees | Traversal, validation, construction, BST recovery |
| Linked Lists | Reverse, rotate, merge, remove nth node |
| Hashing | Two sum, repeated DNA sequence, longest substring |
| Bit Manipulation | OR/AND subset and subarray problems |

## Build and Run

Compile any standalone file with a C++17 compiler:

```bash
g++ -std=c++17 "path/to/file.cpp" -o app
./app
```

On Windows PowerShell:

```powershell
g++ -std=c++17 "path\to\file.cpp" -o app.exe
.\app.exe
```

## Notes

- Most files are standalone practice programs.
- Some LeetCode files are solution snippets and may need the LeetCode-provided class/function wrapper.
- Temporary editor files should not be committed.
